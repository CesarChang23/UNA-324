- Segundo ejercicio, sobre el método de ordenamiento :

    El enunciado del ejercicio da la libertad de utilizar el método de nuestra preferencia.
    El plan de curso menciona: Algoritmo de Burbuja. Algoritmo de Inserción. Algoritmo Shell. 
                               Algoritmo Quicksort. Algoritmo Heapsort. Binsort y Radixsort.
    El plan de curso también considera el método de ordenamiento de Árbol Binario dentro del mismo objetivo de evaluación.
    El método de ordenamiento con árbol binario fue seleccionado porque:
        - Presenta un rendimiento eficiente.
        - Mantiene la estabilidad al no alterar el orden de elementos que son iguales entre sí.
        - No necesita espacio adicional de almacenamiento.
        - Es capaz de ordenar listas en el orden en que se le proporcionan.
